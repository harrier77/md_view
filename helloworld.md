# Hello, md_view!

Questa è una **prova** del visualizzatore Markdown.

## Features

- [x] Lettura file Markdown
- [x] Conversione in HTML con `cmark`
- [x] Visualizzazione con WebKitGTK
- [x] Tema chiaro/scuro automatico

## Esempio di codice

```c
#include <stdio.h>

int main(void) {
  printf("Hello, md_view!\n");
  return 0;
}
```

> "Il Markdown è un linguaggio di markup leggero con sintassi di formattazione del testo semplice."
